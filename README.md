An IRC bot using Node.js and modern JavaScript.

## Things to do

```shell
npm start -- --help
```

Testing uses Mocha with Chai with Nock HTTP mocking.

```shell
npm test
```

This project conforms to [JavaScript Standard Style](https://github.com/feross/standard).

```shell
npm run-script lint
```
